"@matadormq/alerts": minor
---

Add `@matadormq/alerts`: generate Prometheus alert rules (high failure rate, large
backlog, slow processing, paused queue), multi-window multi-burn-rate SLO rules for
wait-time objectives, and a Grafana dashboard JSON with a queue template variable.
Pure generation with no runtime cost; ships a `matador-alerts emit` CLI.
